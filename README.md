# MacathonProject
Feeling nervous in interviews? Calm your nerves with MockCortex, an AI interview coach with professional voice guidance. Get real practice while improving your technical interview skills.

## About
As interviews involve high stakes and are commonly intimidating, MockCortex helps interviewees practice with structured prompts, voice guidance, and actionable feedback. To improve education outcomes, MockCortex aims to fill the gap between schooling and getting a job, where new graduates often have limited interviewing experience.

MockCortex operates as an app that uses Gemini and ElevenLabs to create interview questions and interpret audio replies.

### Libraries
The following are installed in the backend server
```terminal
npm install cors
npm install dotenv --save
npm install express
npm install @google/genai
npm install @elevenlabs/elevenlabs-js
```
